What's a .md File?

Markdown = Universal AI format

.md (Markdown) is the standard format that every AI platform can read. It's not a proprietary file—it's plain text with simple formatting that works everywhere.

Why .md files are perfect for AI skills

Every major AI platform—Claude Code, OpenClaw, Cursor, Codex, Gemini—natively understands Markdown. No conversion needed. No special apps. Just open and use.

# This is a Markdown file (.md)
# It works with EVERY AI platform

# Video Script Framework

## Instructions for the AI:
You are a video script strategist. Follow this
5-beat structure to write high-converting scripts...

## The Framework:
1. **Hook** (0-3 seconds): Grab attention
2. **Problem** (3-8 seconds): Agitate pain
3. **Solution** (8-15 seconds): Introduce product
4. **Proof** (15-25 seconds): Show results
5. **CTA** (25-30 seconds): Call to action
The key difference: You're not reading this document—you're giving it to your AI

Traditional PDFs = you read and learn, then do the work yourself.
Creative Record skills = your AI reads the framework, then does the work WITH you.

Platform-Specific Setup

Getting started takes 2 minutes

Here's exactly how to use your skills with the most popular AI platforms:

◈

Claude Code

Drop the SKILL.md file into your project folder. Claude Code will automatically read it and follow the instructions. Type /help to see available commands from the skill.

Pro tip: Put skills in a /skills folder in your project root. Claude will reference them automatically.
🎯

OpenClaw

Place the SKILL.md file in your workspace. OpenClaw will load it as a context document. Reference the skill by name in your prompts—OpenClaw will apply the framework automatically.

Pro tip: Use the @skill syntax to invoke specific frameworks from your loaded skills.
⚡

Cursor

Open Cursor, go to File → Open Folder, and select your skill folder. The SKILL.md becomes part of your context. Use Cmd+K (Mac) or Ctrl+K (Windows) to ask Cursor to apply the framework.

Pro tip: Add the EXAMPLES/ folder to your workspace so Cursor sees what good output looks like.
💬

ChatGPT / Claude.ai

Open the SKILL.md file in any text editor, select all (Cmd+A), copy (Cmd+C), then paste into your chat. Start with: "Follow this framework:" and paste the content. Save the conversation for reuse.

Pro tip: Create a custom GPT with your skill as the instructions for one-click access.
