## Platform-Specific Notes

### Midjourney Tips
- Use `--style raw` for photographic realism
- Use `--ar` for aspect ratios
- Add `--v 6` if needed for specific version
- Use `--no` for negative prompts

### DALL-E Tips
- Be very specific about colors with hex codes
- Mention "professional photography" for realistic results
- Specify lighting conditions explicitly
- Request "clean white background" for logos

### Leonardo.ai Tips
- Use PhotoReal model for photography
- Use DreamShaper for more stylized results
- Negative prompts work well in the dedicated field

## Color Reference for All Prompts

- **Navy**: #1a1a2e (deep navy blue)
- **Coral**: #e94560 (vibrant coral red)
- **White**: #ffffff
- **Gray**: #f5f5f5 (light gray)
