## How to Score

Rate each item 1–5:
- **5** = Excellent, no changes needed
- **4** = Good, minor improvement possible
- **3** = Average, needs improvement
- **2** = Poor, significant issues
- **1** = Critical failure — fix this first

## Score Key

- **100–125:** Excellent. This ad should be scaled. Make more like it.
- **75–99:** Good. Identify the lowest-scored section and make one change.
- **50–74:** Average. Major revision needed. Start with Section 1 (Hook) first.
- **25–49:** Poor. This ad should be rebuilt from scratch using the Video Script Framework.
